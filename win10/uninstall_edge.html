How to uninstall Win10 Edge browser

C:\Windows\SystemApps\Microsoft.ECApp_8wekyb3d8bbwe is Edge s location.

First,you need to own this folder,

  1. Click the second icon from left on task bar


  2. In the search box type ‘cmd’


  3. Right click,select ‘run as administrator’,


  4. In the command window,


    Run

    1. takeown /f C:\Windows\SystemApps\Microsoft.MicrosoftEdge_8wekyb3d8bbwe
    2. icacls C:\Windows\SystemApps\Microsoft.MicrosoftEdge_8wekyb3d8bbwe /grant administrators:f
    Meaning
    1. Change the owner of the directory to you.
    2. Grant user administrator access to the directory.
Now you can delete or rename the directory,then when you click Edge icon in start menu,
it will not response,but I dont know how to delete the icon in start menu.
