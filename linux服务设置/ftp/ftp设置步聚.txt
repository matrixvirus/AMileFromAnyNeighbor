1 项目介绍
	2018年01月31日，练习设置ftp服务，用本机和一台虚拟机。
	本机系统:openSUSE-Leap-42.3
	虚拟机系统:CentOS-6.9
2 起步步聚，最简单的安装，连接。
2.1. 本机安装vsftpd
	$ zypper install vsftpd
2.2. 虚拟机安装ftp
	$ yum install ftp
2.3 使用叫ftp的客户端
	$ ftp -n
		-n 这个参数作用时等下给你机会输入用户名，而不是直接用你登录虚拟机
		的用户，假如用root登录了虚拟机，执行了ftp，就直接用root登录远程的
		ftp服务了，不能换成别的。
	现在在ftp交互界面里面了，命令行符变成了ftp>。
	1 连接一个ftp服务器
	  open 192.168.100.174
	2 登录
	  user
	3 ls,cd命令和linux系统的一样。
	4 下载一个文件
	  get filename
	5 上传一个文件
	  put filename
    6 下载多个文件
 	  mget file1 file2
	7 上传多个文件
	  mput file1 file2
2.3.1 
	使用主动模式
	$ ftp -A
2.3.2 使用二进制传输模式，如果下载了压缩包，解压时报错，是因为没用二进制转输模式
	。
	ftp> binary
	
3. 怎样下载ftp服务器的一个目录
	ftp命令不能下载目录，最多能用子命令mget下载多个普通文件。
	用lftp或wget两种方法可以下载目录。
3.1 用lftp下载目录
3.1.1 安装lftp
	$ yum install lftp
3.1.2 在命令行输命令lftp
	$ lftp
	进到lftp交互界面后，命令行符变成了lftp >
	连接
	  open ftp服务器地址
    登录
	  user 用户名 密码
	下载文件
	  get filename
    上传文件
	  put filename
    下载多个文件
	  mget file1 file2
	上传多个文件
	  mput file1 file2
	下载目录Music
	  lftp > mirror Music
3.2 用wget下载目录 
	$ wget --mirror ftp://用户名:密码@192.168.100.174/路径/
4 客户端用root用户登录，服务端要设置什么
	刚配的vsftpd只能用普通用户登录，要允许root登录，要改一些设置。
4.1 openSUSE上的步聚
	vim /etc/ftpusers
	有一行是'root',注释掉
	重启服务
	systemctl restart vsftpd
4.2 CentOS上的步聚，和openSUSE上要改的文件，文件具体位置不一样。
	要改两个文件
		/etc/vsftpd/ftpusers
		/etc/vsftpd/user_list
	把是'root'的那一行都注释掉，重启服务
		service vsftpd restart
5 用ftp作为yum源
  格式是这样
	ftp://用户名:密码@地址/路径/
  假如用户在服务器上家目录是/home/opensuse，下面有个文件iso,应该这样写
    ftp://用户名:密码@地址/iso
  直接写iso，下面这样写是错的
    ftp://用户名:密码@地址/home/opensuse/iso
6 SSL/TLS
	编辑配置文件vsftpd.conf,添加这些内容
	ssl_enable=YES
	下面三行是可选的
	ssl_tlsv1=YES
	ssl_sslv2=NO
	ssl_sslv3=NO
